# Vibe Music App

## Project Documentation
The Vibe Music App is a music streaming application that caters to various moods and genres.

## Setup Instructions
1. **Clone the repository:**  
   `git clone https://github.com/yourusername/vibe-music-app.git`
2. **Navigate to the project directory:**  
   `cd vibe-music-app`
3. **Install dependencies:**  
   `npm install`
4. **Run the application:**  
   `npm start`

## Features
- Music streaming across multiple genres.
- Personalized playlists based on user mood.
- Option to create and share playlists.
- User-friendly interface for easy navigation.

## Mood Categories  
- Happy  
- Sad  
- Energetic  
- Relaxed  
- Romantic  

## Bilingual Support
The app offers support for English and Spanish, allowing users to switch languages from the settings menu.
